RECOMMENDED SETUP (Cloud Storage):
═══════════════════════════════════════════════════════════════

STEP 1: On Laptop 1
───────────────────
1. Install Dropbox: https://www.dropbox.com/download
   (Or OneDrive/Google Drive - free accounts work!)

2. Sign in and wait for sync to complete

3. Double-click: setup-cloud-sync.bat

4. When asked for path, enter:
   C:\Users\YourName\Dropbox\KorConstructions-Data

5. Done on Laptop 1!


STEP 2: On Laptop 2
───────────────────
1. Install Dropbox (same account as Laptop 1!)

2. Wait for the folder to sync

3. Copy KorConstructions folder to Laptop 2

4. Double-click: setup-cloud-sync.bat

5. Enter SAME path:
   C:\Users\YourName\Dropbox\KorConstructions-Data

6. Done!


STEP 3: Daily Use
──────────────────
Just use: run-jar.bat

It automatically detects cloud mode!

The window will show:
"[Cloud/Network Mode] Custom data location detected"


═══════════════════════════════════════════════════════════════
IMPORTANT RULES:
═══════════════════════════════════════════════════════════════

⚠️  NEVER run the app on BOTH laptops at the same time!

Correct Workflow:
─────────────────
Laptop 1: Working → Close app → Wait 10 seconds
         ↓
Laptop 2: Can now open app → Work → Close
         ↓
Laptop 1: Can now open app again

Why wait 10 seconds?
- Cloud/network needs time to sync files
- Prevents data conflicts

TESTING YOUR SETUP:
═══════════════════════════════════════════════════════════════

1. On Laptop 1:
   - Start app
   - Add a test customer "Test Sync"
   - Close app
   - Wait 30 seconds

   - Check if "Test Sync" customer appears
